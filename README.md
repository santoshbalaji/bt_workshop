# bt_workshop

### To build the package
``` colcon build ```

### To run example showing BT action calls without using blackboard
``` ros2 run bt_training run_ex_1 ```

### To run example showing BT action calls using blackboard
``` ros2 run bt_training run_ex_2 ```

### To run example showing BT action calls with ROS2 publisher
``` ros2 run bt_training run_ex_3 ```

### To run example showing BT action calls with ROS2 subscriber
``` ros2 run bt_training run_ex_4 ```

### TO run example showing difference between sequence and reactive sequence in BT
``` ros2 run bt_training run_ex_51 ```

``` ros2 run bt_training run_ex_52 ```
